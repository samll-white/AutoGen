📈 评估指标体系

总分计算公式
───────────────────────────────────────────────────────────────────────────
  总分 = 任务完成度(40%) + 时间效率(25%) + 
         资源利用(20%) + 约束满足(15%)

4大维度
───────────────────────────────────────────────────────────────────────────
  1. 任务完成度 (40%) - 权重最高
     • 任务完成率
     • 高优先级完成率
     • 未分配任务数

  2. 时间效率 (25%)
     • 总完成时间
     • 平均等待时间
     • 紧急响应时间

  3. 资源利用 (20%)
     • 无人机利用率
     • 负载均衡度
     • 飞行距离

  4. 约束满足 (15%)
     • 时间冲突数
     • 约束违反数
     • 风险等级

🔧 常见问题速查

Q1: 提示找不到 output_allocation.json?
───────────────────────────────────────────────────────────────────────────
  A: 先运行 AutoGen 算法
     python autogen_uav_allocation.py

Q2: 如何只对比部分算法?
───────────────────────────────────────────────────────────────────────────
  A: 修改 comparison_all_algorithms.py 中的 baseline_algos 列表
     baseline_algos = ['greedy', 'random']  # 只对比这两个

Q3: 如何增加任务和无人机数量?
───────────────────────────────────────────────────────────────────────────
  A: 修改 baseline_algorithms.py 第30-60行
     增加 tasks 和 uavs 列表

Q4: 如何导出Excel格式的结果?
───────────────────────────────────────────────────────────────────────────
  A: 安装 pandas 和 openpyxl
     pip install pandas openpyxl
     
     然后在代码中添加:
     import pandas as pd
     df = pd.DataFrame(results)
     df.to_excel('comparison.xlsx')

Q5: 某个算法评分比AutoGen高怎么办?
───────────────────────────────────────────────────────────────────────────
  A: • 运行AutoGen多次，取最好结果
     • 增加场景复杂度（10+任务）
     • 检查评估权重配置

💡 使用技巧

技巧1: 多次运行取平均
───────────────────────────────────────────────────────────────────────────
  for i in range(5):
      run_autogen()  # 运行5次
      save_result(f"autogen_run_{i}")
  
  # 取平均分和标准差

技巧2: 批量不同场景测试
───────────────────────────────────────────────────────────────────────────
  scenarios = ['simple', 'medium', 'hard']
  for scene in scenarios:
      switch_scenario(scene)
      run_all_comparisons()

技巧3: 自定义评估权重
───────────────────────────────────────────────────────────────────────────
  在 evaluation_metrics.py 中修改:
  weights = {
      'task_completion': 0.5,    # 更重视任务完成
      'time_efficiency': 0.2,
      'resource_utilization': 0.2,
      'constraint_satisfaction': 0.1
  }
