## ✨ 三大核心功能

### 功能1️⃣: AutoGen多智能体任务分配

**文件**：`autogen_uav_allocation.py`

**功能**：
- ✅ 5个专业智能体协作
- ✅ 自动冲突检测
- ✅ 投票决策机制
- ✅ JSON标准输出
- ✅ 自动评估和可视化

**运行**：
```bash
python autogen_uav_allocation.py
# 或双击 run_uav_demo.bat
```

**输出**：
- `output_allocation.json` - 分配方案
- `output_allocation_evaluation.json` - 评估结果
- `visualization_outputs/` - 5张可视化图表

---

### 功能2️⃣: 定量评估系统

**文件**：`evaluation_metrics.py` + `visualize_results.py`

**功能**：
- ✅ 4大维度评估指标
- ✅ 加权评分系统
- ✅ 自动生成评估报告
- ✅ 5种专业可视化图表

**评估指标**：
```
总分 = 任务完成度(40%) + 时间效率(25%) + 
       资源利用(20%) + 约束满足(15%)
```

**运行**：
```bash
python run_evaluation.py output_allocation.json
```

---

### 功能3️⃣: 对比实验系统 ⭐ NEW

**文件**：`baseline_algorithms.py` + `comparison_experiments.py` + `comparison_visualization.py`

**功能**：
- ✅ 4种传统算法实现（贪心、随机、遗传、整数规划）
- ✅ 自动运行和评估
- ✅ 生成对比报告
- ✅ 5种对比可视化图表

**运行**：
```bash
python run_comparison.py
# 或双击 run_comparison.bat
```

**输出**：
- `comparison_results/` - 对比数据
- `comparison_visualizations/` - 5张对比图表

## 🔧 自定义配置

### 修改场景复杂度

编辑 `baseline_algorithms.py` 第30-60行：

```python
@classmethod
def from_default_scenario(cls):
    # 修改任务数量
    tasks = [...]  # 增加或减少任务
    
    # 修改无人机数量
    uavs = [...]  # 增加或减少无人机
    
    # 修改约束条件
    constraints = [...]  # 增加约束
```

### 修改评估权重

编辑 `evaluation_metrics.py` 第290行左右：

```python
weights = {
    'task_completion': 0.4,       # 调整权重
    'time_efficiency': 0.25,
    'resource_utilization': 0.2,
    'constraint_satisfaction': 0.15
}
```

### 添加新的对比算法

1. 在 `baseline_algorithms.py` 添加新类
2. 在 `run_baseline_algorithm()` 注册
3. 重新运行对比实验
